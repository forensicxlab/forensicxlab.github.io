VolWeb Link to heading

VolWeb is a platform that present itself as a web interface to Volatility 3 built using the Django framework. The objectives for the project are to improve investigator efficiency, centralize collaborative analysis, and make memory analysis more “human” friendly. VolWeb also allows investigators to manage memory analysis investigations and search for string-based indicators of compromise. The project can be found here. This project is aimed at incident responders that often needs to work fast and gain time to quickly identify threats and indicator of compromise and produce quick technical reports.
