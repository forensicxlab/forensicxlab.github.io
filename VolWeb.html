VolWeb

A Digital memory forensic tool for incident responder

What is memory analysis ?

Memory is focusing on the extraction of evidence by exploiting the Random Access Memory (abbreviated to RAM) of a given system. The memory has much better performances than a classical hard drive. Indeed, because of the quick computing capacities of a CPU, all the instructions that needs to be ran are meant to be stored into a high transfer rate memory device. RAM can store every object that are needed in order for a program to execute (files, logs, network communications, etc). For example, a program reading a file will result into the file being load into memory. RAM also interact with the hard drive to store data to use later.

If you want to learn more about memory analysis I recommend reading this book.

Volatility & VolWeb

Volaltility is the best framework to perform digital forensic on memory. Volatility has become the world’s most widely used memory forensics platform and is currently on active development. It is an open-source project that can be found and downloaded here.

VolWeb is a platform that present itself as a web interface to Volatility 3 built using the Django framework. The objectives for the project are to improve investigator efficiency, centralize collaborative analysis, and make memory analysis more "human" friendly. VolWeb also allows investigators to manage memory analysis investigations and search for string-based indicators of compromise. The project can be found here. This project is aimed at incident responders that often needs to work fast and gain time to quickly identify threats and indicator of compromise and produce quick technical reports.

Features

VolWeb comes with a lot of features to facilitate memory forensics.
Automated analysis

VolWeb provide the possibility to automatically analyse, extract, parse and display forensic artifacts from a windows memory dump.




Work together

Each investigation can be viewed and analysed by separated analysts that have an account on the platform. 




Generate reports

Each artifact can be tagged and aggregated into a report that can be used or integrated to a investigation report.
