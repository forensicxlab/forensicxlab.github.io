Linux Analysis

In this example, the user is uploading a memory image of a Linux machine and is binding to this case a ISF containing the kernel symbol tables. With this the VolWeb engine is able to parse this memory image and extract data.

Windows Analysis

In this example, the user is uploading a memory image of a Windows machine and is binding to this case a string based IOC. With this the VolWeb engine is able to look for this IOC in the memory image and extract the related PIDs.
