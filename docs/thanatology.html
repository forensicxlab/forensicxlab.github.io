游불 Thanatology (in progress)

Thanatology is built using the Tauri framework to deliver a cross-platform digital forensics desktop application. It leverages the power of the Exhume toolkit as a core library and uses React with MUI to showcase how a modern application tailored for law enforcement can be developed.

You can find the project developpement repository and roadmap here: https://github.com/forensicxlab/thanatology
